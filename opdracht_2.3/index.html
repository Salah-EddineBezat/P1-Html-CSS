Onderzoek

• Waarom noem je de eerste HTML-pagina; index.html?

Antwoord: Om ervoor te zorgen dat een browser herkent dat dit een “HTMl” pagina is

• Met welke HTML versie werken we nu?

Antwoord: Momenteel werken we met de versie HTML 5.

• Met welke CSS versie werken we nu?

Antwoord: We werken met de versie CSS 3, de nieuwste versie van CSS. Deze versie bied nieuwe functies en verschillende verbeteringen in de stylemogelijkheden.

• Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Antwoord: Op een andere browser ziet een website er net iets anders uit, omdat elke browser webcodes op zijn eigen manier in beeld.

• Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Antwoord: De Google Chrome browser is de populairste browser en verbruikt momenteel rond de 1 GB RAM.
